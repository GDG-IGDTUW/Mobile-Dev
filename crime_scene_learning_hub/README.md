Crime Scene Learning Hub is an open-source Flutter application that helps users learn **complex technical and system concepts** through a **crime investigation metaphor**.  
Instead of traditional explanations, each topic is presented as a **case** with suspects, evidence, and a final verdict‚Äîmaking learning more engaging, intuitive, and memorable.

Project Maintainer: **[Yashvi Gupta](https://github.com/YashviGupta191105)**

---

## Concept

Many learners struggle with abstract topics like deadlocks, system failures, or design flaws.  
Crime Scene Learning Hub simplifies these concepts by treating them as **investigation cases**, where users analyze clues, identify suspects, and reach logical conclusions.

---

## App Structure

The app is divided into **three main sections**:

### 1. Explore Cases üîç

This section focuses on **learning and understanding concepts**.

**What it includes:**
- A list of case files (e.g., Deadlock in Multithreading, Deadlock in Databases, etc.)
- Category-based filtering (OS, DBMS, CN, etc.)
- Each case is structured as:
  - **Suspects** ‚Äì possible causes or components involved  
  - **Evidence** ‚Äì symptoms, observations, and technical clues  
  - **Final Verdict** ‚Äì clear explanation of the root cause and solution  

This storytelling approach helps users understand *why* a problem happens, not just *what* it is.

---

### 2. Solve a Case üß©

This section focuses on **testing understanding**.

**How it works:**
- Users are given a case-based quiz
- Each case contains **4‚Äì5 questions**, such as:
  - Identify the suspect  
  - Find the system flaw  
  - Choose the correct execution/order  
- Questions are evaluated as **right or wrong**
- Helps reinforce learning through application

---

### 3. Profile Section üë§

- Track cases explored and solved
- General FAQ's about the app
