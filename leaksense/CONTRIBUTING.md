# Contributing to LeakSense

Thank you for your interest in contributing to LeakSense!

This project is currently focused on improving UI, UX, and overall Flutter app structure. You do not need any domain knowledge about IoT or hardware to contribute.

## How to Contribute
1. Fork the repository
2. Create a new branch for your change
3. Pick an issue from the issue list
4. Make your changes following existing code style
5. Open a pull request with a clear description

## Guidelines
- Keep UI clean and consistent
- Prefer reusable widgets where possible
- Avoid adding backend or hardware integrations
- Keep changes scoped to the issue you’re working on

## Beginner Friendly
Issues labeled as beginner or UI-focused are great starting points if you’re new to Flutter or open source.

Happy building!
