# LeakSense

LeakSense is a Flutter-based application designed to help users monitor and understand water leakage inside their homes. The app presents leak information in a room-wise and visually intuitive manner, making it easier for users to identify affected areas and take timely action.

Rather than focusing on complex infrastructure or hardware integration, this repository currently emphasizes **user experience, UI design, and app structure**, serving as a solid foundation for future extensions.

## Features
- Dashboard overview of system status
- Room-wise leak visualization
- Leak insights with estimated risk and severity
- Historical view of detected leaks
- Tips and actions for handling leaks
- Settings and profile management

## Project Status
This is an early-stage version focused on UI and interaction design. Data shown in the app is currently mocked to demonstrate layout and flow.

## Tech Stack
- Flutter
- Material 3
- Dart
