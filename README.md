# Mobile-Dev-Repo 

A curated collection of Flutter-based mobile applications built by the Mobile Development community. Each project addresses a unique real-world use case while showcasing different aspects of mobile app development such as UI/UX design, AI integration, storytelling, and intelligent systems.

This repository is ideal for developers who want to explore, learn, and contribute to meaningful mobile projects with varying levels of complexity.

## Projects Overview

### 1. AfterHours – Daily Reflection App

**Description:**  
AfterHours is a Flutter-based reflection app that helps users talk about their day through simple, story-style prompts. Instead of free-form journaling, the app breaks the day into gentle narrative moments, making self-expression easier and less overwhelming.

**Features:**
- Guided story-based daily reflection prompts  
- Calm, minimal UI focused on mental well-being  
- Structured flow to reduce journaling anxiety  

---

### 2. Crime Scene Learning Hub

**Description:**  
Crime Scene Learning Hub is an open-source Flutter application that explains complex concepts using a crime-investigation metaphor. Each topic is presented as a case file with suspects, evidence, and a verdict to help learners understand causes, effects, and solutions clearly.

**Features:**
- Case-based learning using investigation metaphors  
- Topics structured as suspects, evidence, and conclusions  
- Engaging and intuitive UI  

---

### 3. LeakSense – Smart Water Leak Detection

**Description:**  
LeakSense is a Flutter-based consumer application designed to detect, localize, and explain water leakage in household and commercial pipeline systems. It combines sensor-driven signals with intelligent inference to help users prevent water loss and structural damage.

**Features:**
- Water leak detection using sensor data  
- Estimation of leak location and duration  
- Timely alerts and actionable insights  

---

### 4. SplitSmart – Smart Bill Splitting App

**Description:**  
SplitSmart is a Flutter-based bill-splitting app with AI-powered receipt scanning and a modern UI. It simplifies group expenses by splitting bills accurately, calculating tips, and enabling quick sharing via WhatsApp.

**Features:**
- AI-powered receipt scanning  
- Accurate bill splitting among friends  
- Built-in tip calculator  
- WhatsApp sharing support

## Contributing Guidelines

We welcome contributions to all projects in this repository.

### How to Contribute

1. Explore open issues in the project folders.  
2. Comment on an issue to claim it.  
3. Fork the repository and create a new branch.  
4. Make your changes and commit with clear messages.  
5. Push your branch and submit a Pull Request (PR).  

---

## Steps to Get Started

1. Browse the projects and select one to contribute to.  
2. Check existing issues or create a new one.  
3. Tag the maintainers if you face any issues.  
4. Wait for the issue to be assigned before starting.  
5. Fork the repository and work on a dedicated branch.  
6. Submit a PR mentioning the issue number and changes made.  
7. Attach screenshots or screen recordings to your PR.  
8. Follow the contribution guidelines mentioned in each project’s README.
